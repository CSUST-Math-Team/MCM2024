连接至服务器

Web 报告与因特网服务器的连接状态将显示在 Web 报告的右上角。
打开 Web 报告无需因特网连接。 借助 Web 报告的已保存副本，您可以查看所有图表和表，并在未连接到因特网时具有一些受限编辑能力。
使 Web 报告连接到因特网服务器将对表和图表启用更多编辑功能。
  • 如果作者在创建 Web 报告时指定了因特网服务器，那么 Web 报告将在打开后尝试连接到该服务器。
  • 如果 Web 报告未连接到服务器，请单击服务器“状态消息”以打开用于重试连接、尝试连接到其他服务器或输入新的服务器地址的工具。
  • 有关向因特网服务器添加增强控件的信息，请转到https://developer.ibm.com/predictiveanalytics.
  • 如果指定新服务器连接，那么首选格式为 http://xxx.xxx.xxx.xxx:xxxx。
